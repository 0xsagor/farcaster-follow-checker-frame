# Farcaster Follow Checker Frame

A high-performance, expert-level Farcaster Frame. This repository provides a "flat" structure for deploying a functional Frame that interacts with the Farcaster Hub to check user follow status.

## Features
* **Frog.js Integration:** Utilizes the industry-standard framework for Farcaster Frames.
* **Hub Interaction:** Real-time verification of social graphs via Neynar or public Hubs.
* **Dynamic UI:** Responsive OpenGraph images generated on-the-fly based on user interaction.

## Setup
1. **Environment:** Create a `.env` file with your `NEYNAR_API_KEY`.
2. **Install:** `npm install`
3. **Dev:** `npm run dev`
4. **Deploy:** Optimized for Vercel or any serverless provider.
